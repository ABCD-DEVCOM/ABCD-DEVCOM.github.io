# My Account (MyABCD)

Registered library users can log in to perform self-service tasks without visiting the circulation desk.

## Logging In
1.  Click **Sign In** (or "Ingresar") in the top right corner.
2.  Enter your **User ID** (e.g., Student Number) and **Password**.
3.  *Note:* If you don't have a password, contact the library staff.

## Dashboard Features
Once logged in, you access a personalized dashboard:

### 1. My Loans (Renewals)
* View a list of all items you currently have checked out.
* **Due Date:** Check when books are due to avoid fines.
* **Renew:** If allowed by library policy, a **Renew** button appears next to eligible items. Click it to extend the loan period immediately.

### 2. My Reservations
* View the status of items you have reserved.
* **Cancel:** If you no longer need an item, you can cancel the reservation here to let the next person in line have it.
* **Position:** See your place in the waiting queue (e.g., "2nd in line").

### 3. History & Fines
* **Loan History:** View a log of everything you have read in the past.
* **Fines Statement:** Check if you have any outstanding debts with the library.

### 4. Profile Management
* **Change Password:** You can update your login password for security.
