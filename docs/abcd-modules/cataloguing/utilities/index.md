# Cataloguing Utilities

The **Utilities** menu in the Cataloging module provides tools for batch processing and data exchange.

## Data Tools

* **[Import & Export](import-export.md)**: Move data in and out of ABCD using the ISO 2709 standard. Essential for migrating data between systems.
* **[Global Changes](global-changes.md)**: Perform "Find and Replace" operations, add fields, or delete fields across multiple records simultaneously.

## Maintenance Tasks

Some maintenance tasks accessible from this menu are technical operations shared with the Administration module.

* **Unlock Records / Databases**: If you encounter a "Record Locked" message.
* **Inverted File Generation**: If search results are missing or incorrect.

:::info Administration Guide
For detailed instructions on these maintenance procedures, please refer to the **[Database Maintenance](../../../abcd-administration/maintenance.md)** section in the Administration guide.
:::
