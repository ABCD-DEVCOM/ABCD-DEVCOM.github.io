# Sources on GitHub

The latest version of ABCD is available on the official GitHub organization.

## ABCD-DEVCOM
[https://github.com/ABCD-DEVCOM/ABCD](https://github.com/ABCD-DEVCOM/ABCD)

This repository includes:
* The **source code** of ABCD itself (PHP/Scripts).
* Valid versions for both **Windows** and **Linux**.
* **Executables** (CISIS Utilities) for Windows/Linux.
* **Example databases** compatible with both platforms.

## Community & Updates
For updates, issues, and forks, please refer to the repository's "Issues" tab or the community wiki.
