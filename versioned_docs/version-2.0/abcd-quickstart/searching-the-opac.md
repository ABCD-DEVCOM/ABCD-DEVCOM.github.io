# Guia de Início Rápido: Realizando uma Busca no OPAC

### O que é o OPAC?

OPAC é a sigla para **Catálogo Online de Acesso Público**. É a interface web que os usuários da sua biblioteca (alunos, pesquisadores, público geral) utilizam para pesquisar o acervo.

No universo ABCD, esta interface pública é frequentemente chamada de **iAH** e é integrada ao **ABCD Site**, que funciona como o portal principal da biblioteca. Através dele, os usuários podem encontrar livros, artigos, teses e outros materiais que foram catalogados no sistema.

### Passo 1: Acessar a Interface de Busca

Para realizar uma busca, o usuário deve primeiro acessar o portal público da biblioteca. O endereço será algo como `http://endereço-da-sua-biblioteca.com/opac`.

Na página principal, a ferramenta de busca estará em destaque, geralmente com o título "Pesquisa na ABCD" ou similar.

### Passo 2: A Busca Simples (por Palavras)

A busca simples é a forma mais rápida de encontrar um item e funciona de maneira muito parecida com os buscadores da web.

1.  **Digite os termos:** No campo de texto principal (geralmente com a instrução "Entre com uma ou mais palavras"), digite o título, autor ou assunto que deseja encontrar.
2.  **Escolha o método:**
    * **Todas as palavras:** Retorna apenas resultados que contenham *todos* os termos que você digitou (equivalente a uma busca com "E").
    * **Qualquer palavra:** Retorna resultados que contenham *pelo menos um* dos termos que você digitou (equivalente a uma busca com "OU").
3.  **Clique em "Pesquisar"**: O sistema exibirá uma lista com os resultados encontrados.

### Passo 3: A Busca Avançada (Filtro de Pesquisa)

Para pesquisas mais específicas, você pode usar a busca avançada.

1.  Clique no link **"filtro de pesquisa"** ou **"busca avançada"**, geralmente localizado abaixo do botão de pesquisa principal.
2.  Uma nova tela será exibida, permitindo que você pesquise em campos específicos, como:
    * Título
    * Autor
    * Assunto
    * Editora
3.  Você pode combinar diferentes campos para refinar ainda mais a sua busca.

### Passo 4: Visualizando os Resultados

Após realizar a busca, o sistema apresentará uma lista de registros que correspondem aos seus critérios.

* A lista geralmente mostra informações essenciais, como o título e o autor de cada item.
* Para ver todos os detalhes de um item, como o resumo, número de chamada e disponibilidade, basta clicar no título do registro desejado.
