# Guia de Início Rápido: Criando seu Primeiro Catálogo

Este guia irá orientá-lo através dos passos essenciais para criar um novo catálogo (base de dados) no ABCD e importar seus primeiros registros.

### 1. Acesse o Módulo de Administração

Após fazer o login no ABCD, você estará na tela principal. [cite_start]Na seção **Administração**, clique em **Criar Base de Dados**[cite: 5730].

### 2. Defina as Informações da Nova Base de Dados

Você será direcionado para uma tela onde deve preencher as seguintes informações:

* **Nome da base de dados:** Insira um nome curto e sem espaços. Por exemplo, `meucatalogo`.
* **Descrição:** Dê um nome descritivo que aparecerá nas listas. Por exemplo, `Meu Primeiro Catálogo`.
* **Criar desde:** Para este guia, vamos usar uma estrutura pré-definida. [cite_start]Selecione **MARC** na lista suspensa[cite: 5724, 5736].

[cite_start]Clique em **Continuar**[cite: 5736]. O sistema irá processar a criação e mostrará uma lista de todas as ações realizadas. Ao finalizar, clique em **Regresar** para voltar ao menu principal.

### 3. Importando Registros para o Catálogo

Agora que a base de dados está criada, vamos adicionar registros a ela.

1.  Na tela principal, selecione o seu recém-criado `Meu Primeiro Catálogo` na lista de bases de dados.
2.  [cite_start]Clique em **Entrada de Dados**[cite: 5777].
3.  [cite_start]Na barra de ferramentas superior, clique no ícone de **Utilitários** e, no menu que aparece, selecione **Importar ISO**[cite: 5790, 5791].
4.  Na tela de importação:
    * Clique em **Examinar** (ou "Browse") e localize o seu arquivo de registros no formato ISO (`.iso` ou `.mrc`).
    * [cite_start]Marque a opção **Gerar lista Invertida**[cite: 5800].
    * Clique no botão **Seleccionar** para confirmar o arquivo que você subiu.
    * O sistema pedirá uma confirmação para iniciar a importação. [cite_start]Clique em **Sim**[cite: 5802].

O sistema exibirá o progresso da importação, mostrando os registros à medida que são processados. Ao final, você terá seus primeiros registros catalogados no ABCD!
